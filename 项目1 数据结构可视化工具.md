- ## **项目1: 数据结构可视化工具**

  ### **项目概述**

  开发一个命令行工具，将常见数据结构（数组、链表、栈、队列）以ASCII字符可视化的形式展示出来，帮助理解数据结构的操作过程。

  ### **详细技术需求**

  #### **1.1 基础架构要求**

  \- **语言版本**: Go 1.19+

  \- **项目结构**: 

  ```Plain
  data-visualizer/
  ├── main.go              # 程序入口
  ├── structures/          # 数据结构包
  │   ├── array.go         # 数组实现
  │   ├── linkedlist.go    # 链表实现
  │   ├── stack.go         # 栈实现
  │   └── queue.go         # 队列实现
  ├── visualizer/          # 可视化包
  │   ├── ascii.go         # ASCII绘制工具
  │   └── formatter.go     # 格式化工具
  ├── cli/                 # 命令行界面
  │   └── interface.go     # 交互处理
  └── go.mod
  ```

  #### **1.2 数组(Array)实现要求**

  ```Go
  type Array struct {
      data     []int
      capacity int
      size     int
  }
  
  // 必须实现的方法
  func (a *Array) Insert(index, value int) error
  func (a *Array) Delete(index int) error
  func (a *Array) Get(index int) (int, error)
  func (a *Array) Set(index, value int) error
  func (a *Array) Find(value int) int
  func (a *Array) Display() string
  func (a *Array) Size() int
  ```

  **可视化要求**:

  ```Plain
  数组可视化示例:
  ┌─────┬─────┬─────┬─────┬─────┐
  │  1  │  2  │  3  │     │  5  │
  └─────┴─────┴─────┴─────┴─────┘
    [0]   [1]   [2]   [3]   [4]
  
  操作动画 (插入操作):
  插入前: [1][2][3][ ][5]
          ↑
         pos 1
  插入后: [1][4][2][3][ ]
               ↑
            new element
  ```

  #### **1.3** **链表****(LinkedList)实现要求**

  ```Go
  type Node struct {
      data int
      next *Node
  }
  
  type LinkedList struct {
      head *Node
      size int
  }
  
  // 必须实现的方法
  func (l *LinkedList) InsertHead(value int)
  func (l *LinkedList) InsertTail(value int)
  func (l *LinkedList) InsertAt(index, value int) error
  func (l *LinkedList) DeleteHead() error
  func (l *LinkedList) DeleteTail() error
  func (l *LinkedList) DeleteAt(index int) error
  func (l *LinkedList) Find(value int) int
  func (l *LinkedList) Display() string
  func (l *LinkedList) Size() int
  ```

  **可视化要求**:

  ```Plain
  链表可视化示例:
  HEAD -> ┌───┬───┐ -> ┌───┬───┐ -> ┌───┬───┐ -> NULL
          │ 1 │ ● │    │ 2 │ ● │    │ 3 │ ● │
          └───┴───┘    └───┴───┘    └───┴───┘
  
  简化版本:
  [1] -> [2] -> [3] -> NULL
  
  操作动画 (删除节点):
  删除前: [1] -> [2] -> [3] -> NULL
  删除节点2:   ↑
  删除后: [1] ---------> [3] -> NULL
  ```

  #### **1.4 栈(Stack)实现要求**

  ```Go
  type Stack struct {
      data []int
      top  int
  }
  
  // 必须实现的方法
  func (s *Stack) Push(value int)
  func (s *Stack) Pop() (int, error)
  func (s *Stack) Peek() (int, error)
  func (s *Stack) IsEmpty() bool
  func (s *Stack) Size() int
  func (s *Stack) Display() string
  ```

  **可视化要求**:

  ```Plain
  栈可视化示例:
       栈顶 -> ┌─────┐
               │  3  │ <- 最后入栈
               ├─────┤
               │  2  │
               ├─────┤
               │  1  │ <- 最先入栈
       栈底 -> └─────┘
  
  Push操作动画:
  Push(4)前:          Push(4)后:
  ┌─────┐            ┌─────┐
  │  3  │ <- top     │  4  │ <- new top
  ├─────┤            ├─────┤
  │  2  │            │  3  │
  ├─────┤            ├─────┤
  │  1  │            │  2  │
  └─────┘            ├─────┤
                     │  1  │
                     └─────┘
  ```

  #### **1.5 队列(Queue)实现要求**

  ```Go
  type Queue struct {
      data  []int
      front int
      rear  int
      size  int
  }
  
  // 必须实现的方法
  func (q *Queue) Enqueue(value int)
  func (q *Queue) Dequeue() (int, error)
  func (q *Queue) Front() (int, error)
  func (q *Queue) IsEmpty() bool
  func (q *Queue) Size() int
  func (q *Queue) Display() string
  ```

  **可视化要求**:

  ```Plain
  队列可视化示例:
  出队 <- ┌─────┬─────┬─────┬─────┐ <- 入队
          │  1  │  2  │  3  │  4  │
          └─────┴─────┴─────┴─────┘
         front                rear
  
  操作动画:
  Enqueue(5):
          ┌─────┬─────┬─────┬─────┬─────┐
          │  1  │  2  │  3  │  4  │  5  │
          └─────┴─────┴─────┴─────┴─────┘
         front                     rear
  
  Dequeue():
                 ┌─────┬─────┬─────┬─────┐
                 │  2  │  3  │  4  │  5  │
                 └─────┴─────┴─────┴─────┘
                front               rear
  ```
